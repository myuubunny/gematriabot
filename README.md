# gematriabot
Discord bot for gematria numerology using the pygematria library and other gimmick commands (GematriaBot#9721)
## Commands
* /a: Converts letters to numbers and shows sum in Agrippa cipher
* /eq: Converts letters to numbers and shows sum in English Qabbala cipher
* /fortune: randomly picks an s4s fortune
* /g: Converts letters to numbers and shows sum in Greek isopsephy cipher
* /h: Converts letters to numbers and shows sum in Hebrew cipher
* /r: Converts letters to numbers and shows sum in Rudolff cipher
* g!sync: Syncs bot command tree (owner only)
